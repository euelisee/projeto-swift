# Swift — Projeto Front-End
Este é um projeto front-end desenvolvido como parte do **Challenge FIAP** para a **Swift**, aplicando conhecimentos em classe para trazer soluções reais e práticas de redução de atrito na jornada de compras online.
---
## Tecnologias utilizadas
- HTML5
- CSS3 (com tokens de design)
- Bootstrap 5
- Google Fonts (Barlow Condensed & Inter)
---
## Estrutura do projeto
```bash
/
├── css/
│ ├── bootstrap.min.css
│ ├── style.css # estilos principais
│ └── tokens.css # variáveis e tokens de design
├── pages/ #paginas do protótipo
│ ├── .html
├── img/ # imagens do projeto
├── js/
│ ├── bootstrap.bundle.min.js
│ └── main.js
├── index.html # página inicial
└── README.md
```

---
## Como executar
1. Abra o arquivo `index.html` diretamente no navegador 
---
## Funcionalidades
- Header com busca, categorias e seletor de loja
- Banners promocionais com botões clicáveis
- Seção “Seleção de Produtos” com carrossel
- Cards de receitas com CTA
- Seção “Churrasco no Ponto” com destaques
- Footer com informações de contato e botões clicáveis
---
## Layout
O layout foi baseado no material oficial fornecido pela **Swift**, adaptado para web responsiva.
